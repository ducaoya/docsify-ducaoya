## 小知识点

_毒草鸭，又名毒草呀。是中国中部地区长江流域的一个稀有物种，目前发现现存的毒草鸭仅这里一只。关于它传奇的故事，日后有机会再详说，多多关注它的动态，你就会发现它的神奇之处。_
